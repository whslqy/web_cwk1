Coursework API Demo

Book API control panel

This Django REST Framework project provides a database-backed book catalogue with full CRUD operations, filtering, genre recommendations, aggregate statistics, Swagger documentation, and a browsable API for live demonstrations.

Documentation

Swagger UI

Interactive API documentation generated from the live schema.

Browsable API

Books endpoint

Use DRF's built-in interface to create, view, edit, and delete book records.

Administration

Django admin

Manage the dataset through the admin dashboard with search and filtering.

Key endpoints

The project exposes the minimum CRUD routes plus richer query and recommendation endpoints for the coursework demonstration.

GET /api/books/ List all books, optionally filtered by genre, author, published_year, search, and ordering.
POST /api/books/ Create a new book.
GET /api/books/{id}/ Retrieve one book by ID.
PUT /api/books/{id}/ Replace a book record.
DELETE /api/books/{id}/ Delete a book.
GET /api/books/?genre=Fantasy Filter books by genre in the main collection endpoint.
GET /api/books/stats/ Return collection-wide summary statistics.
GET /api/books/recommendations/?genre=Science%20Fiction&limit=5 Return up to the requested number of matching recommendations from the public dataset.
GET /api/schema/ OpenAPI schema in JSON format.
